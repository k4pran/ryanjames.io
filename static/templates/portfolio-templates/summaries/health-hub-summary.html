A parser for parsing apple health data and REST service to make it available in different formats. It will save parsed data in a sqlite server and give you access to the data via REST endpoints. The endpoint /endpoints will list available endpoints to query and the data can be fetched in different formats such as json, html, csv or pandas dataframe via pickle. The content returned is controlled by the content-type header in the http request.
