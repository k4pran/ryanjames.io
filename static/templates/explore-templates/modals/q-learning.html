In reinforcement learning we want an agent to learn the optimal actions $a$ to perform in a state $s$ in order to maximize a reward $r$. Q-learning is a learning method that can approximate how valuable performing a given action in a given state is, and use this approximation to make optimal decisions. In Q-learning the agent learns by experience and is categorized as model-free. This means that it doesn't require a model that says - given my current state $s$ what is the probability of transitioning to another state $s_1$ and receiving an associated reward if I take a given action $a$.
