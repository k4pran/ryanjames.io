Oak

  • java
  • maven

Oak is a way of creating ocarina video and PDF tutorials. It takes a midi file and maps the notes to ocarina tabs and processes them into a complete tutorials.

Features
  • Outputs an mp4 and/or a PDF with ocarina tabs.
  • Packaged with an ocarina soundfont to output realistic audio
  • Add your own custom background
  • Frame rate and audio offset can be set to fine-tune video/audio sync.
  • Add custom text to introduce the video, useful for including website links and contact info.
